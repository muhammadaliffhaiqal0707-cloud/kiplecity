# KipleCity Website

This is a static website converted from `KipleCity Corporate Deck V3.pdf`.

## Files
- `index.html` - main website
- `styles.css` - full responsive styling
- `script.js` - mobile menu + slide gallery lightbox
- `assets/slides/` - all 22 deck pages extracted as images

## How to run
Open `index.html` in any browser, or host the folder on Netlify, Vercel, cPanel, or any static hosting provider.
